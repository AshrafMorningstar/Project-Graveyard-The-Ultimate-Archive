# Local Events Offline PWA

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A Progressive Web App for discovering local community events with offline-first architecture. Features service worker caching, IndexedDB sync, and installable app experience.

## How to Use

1. Open `index.html` in a modern browser
2. Click "Install" to add to your device
3. Browse events in List, Map, or Calendar view
4. Click hearts to favorite events
5. Works offline after first visit
