# MentorMatch PWA

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A Progressive Web App (PWA) facilitating instant micro-mentorship sessions. It bridges the gap between professionals with a mobile-first interface and simulated WebRTC video calling capabilities.

## How to Try

1. Open `index.html` in Chrome or Edge.
2. Click the specific mentor's "Instant Session" button.
3. Grant notification permissions (simulated).
4. Experience the video call overlay with mute/camera controls.
