# Micro Portfolio Generator

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A zero-config single-page application that generates polished portfolio websites from a simple form. Features drag-and-drop section ordering, three curated premium templates, and instant preview with static site export.

## Live Demo

Open `index.html` in your browser to start creating portfolios instantly.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox
- **Vanilla JavaScript** - No dependencies
- **LocalStorage** - Auto-save functionality

## How to Use

1. Fill in your personal information in the left panel
2. Add your skills (comma-separated)
3. Include social media links
4. Choose from three premium themes
5. Watch the live preview update in real-time
6. Click "Export Portfolio" to download your site
