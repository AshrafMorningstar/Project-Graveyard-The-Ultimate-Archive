# Contributing to Web-dev-mini-projects

This repository aims to be the **#1 collection of Premium Web Development Mini-Projects**. We value quality, uniqueness, and modern design over simple copy-paste tutorials.

## üé® Design Standards (CRITICAL)

All submitted projects **MUST** adhere to one of the following premium design styles:

1. **Glassmorphism**: Translucent backgrounds, blur, vivid colors.
2. **Neumorphism**: Soft shadows, non-flat UI.
3. **Cyberpunk/Holographic**: Neon, dark mode, futuristic.
4. **Minimalist Luxury**: High-end typography, whitespace.

**‚ùå We do NOT accept:**

- Plain HTML default styles.
- Bootstrap default look.
- Projects without custom CSS.

## üõ† How to Contribute

1. **Fork** the repository.
2. **Create a Branch**: `git checkout -b feature/AmazingProject`
3. **Select a Project**: Check the `project_manifest.json` or pick an empty folder.
4. **Implement**:
   - Use `index.html`, `style.css`, `script.js`.
   - Ensure it is responsive.
   - Add your name to the `meta` author tag.
5. **Commit**: `git commit -m "Add AmazingProject"`
6. **Push**: `git push origin feature/AmazingProject`
7. **Open a Pull Request**.

## üìù Code Guidelines

- Use semantic HTML5.
- Use clean, commented JavaScript (ES6+).
- Do not include `node_modules`.
