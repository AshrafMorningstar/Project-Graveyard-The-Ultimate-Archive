## üíª Usage

1. **Clone the repository**

```bash
git clone https://github.com/AshrafMorningstar/QuantumFlow.git
cd QuantumFlow
```

2. **Open in browser**

```bash
# Simply open index.html in your favorite browser
# No build process required!
```

3. **Interact**

- Click "Switch Theme" to cycle through themes
- Use "Mode" to toggle display modes
- Click "Generate Visualization" to create new patterns
- Adjust sliders to control visualization parameters

## üé® Customization

### Themes

Edit the CSS variables in `style.css`:

```css
:root {
  --primary: #00f0ff;
  --secondary: #ff00ff;
  --accent: #ffff00;
}
```

### Visualizations

Modify the visualization algorithms in `script.js`:

```javascript
neuralNetwork(index);
quantumStream(index);
holographicMatrix(index);
dimensionalAnalysis(index);
```
