### GitHub Actions

To use this in your own repository, create `.github/workflows/stats.yml`:

```yaml
name: Generate Cyber Stats
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: AshrafMorningstar/cyber-stats-remotion@v1
        with:
          config_file: "cyber-config.yml"
          github_token: ${{ secrets.GITHUB_TOKEN }}
      - uses: stefanzweifel/git-auto-commit-action@v4
        with:
          commit_message: "âœ¨ Update cyber stats"
```
