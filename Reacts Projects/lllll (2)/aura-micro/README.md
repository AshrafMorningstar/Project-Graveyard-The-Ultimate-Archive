**Aura Micro** is a compact, delightful generator that produces a different, small, elegant composition every render using a micro-theme engine.

## Features

- **Micro-Theme Engine**: Generates one of dozens of palettes/layouts based on a seed.
- **Lightweight**: Minimal dependencies, fast render.
- **Unique Output**: Every seed produces a distinct look.

## Usage

1. `npm install`
2. `npm run render` -> Outputs `out/aura.gif`
