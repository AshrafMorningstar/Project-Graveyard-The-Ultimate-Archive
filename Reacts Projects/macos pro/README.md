# MacOS Portfolio

A macOS-style portfolio built with React, TypeScript, Tailwind CSS, GSAP, and Zustand.

## Features

- **Interactive Dock**: Smooth magnification animations using GSAP.
- **Window Management**: Draggable, resizable, minimizable, and maximizable windows using `react-rnd` and Zustand.
- **Global State**: Managed with Zustand for z-indexing, open/active apps.
- **Apps**:
  - **About Me**: Personal information.
  - **Safari**: Browser simulation (points to GitHub).
  - **Terminal**: Interactive-looking terminal.
  - **Finder**: File system placeholder.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run development server:

   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Animations**: GSAP
- **State Management**: Zustand
- **Icons**: Lucide React
