# TITAN REALITY WEAVER

**Titan Reality Weaver** is a system package for differentiable hybrid simulations.

## Deliverables

1. **[Docs] System Architecture**: See `docs/architecture.md`
2. **[Docs] Composition DSL**: See `docs/dsl_spec.md`
3. **[Code] Minimal Runtime Scaffold**: See `runtime/`
   - Rust Core: `runtime/runtime_core`
   - Python Bindings: `runtime/py_bindings`
4. **[Docs] OpenAPI Spec**: See `docs/openapi.yaml`
