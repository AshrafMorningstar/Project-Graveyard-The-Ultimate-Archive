- `input.json`: Data source for the video.

## How to Run Locally

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Preview Video**:
    ```bash
    npm start
    ```
    Opens the Remotion Player in your browser.

3.  **Render Video**:
    ```bash
    npm run build
    ```
    Outputs `out/video.mp4`.
