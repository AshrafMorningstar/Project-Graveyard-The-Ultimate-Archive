## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/AshrafMorningstar/CollabDev_Studio.git

# Navigate to project directory
cd CollabDev_Studio

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173` to see the application.
