# macOS-style Developer Portfolio â€” Ashraf Morningstar

A macOS-style interactive developer portfolio built with React, Tailwind CSS, GSAP, and Zustand.

## Tech Stack

- **Framework**: React 18 (Vite)
- **Styling**: Tailwind CSS
- **Animations**: GSAP
- **State Management**: Zustand
- **Icons**: Lucide React

## Features

- Interactive Dock with hover animations
- Window management (Drag, Resize, Minimize, Toggle) - _In Progress_
- Music Player - _Planned_
- Project Showcase - _Planned_

## Quick Start

1.  **Clone the repository**

    ```bash
    git clone https://github.com/AshrafMorningstar/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Run localized development server**

    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
