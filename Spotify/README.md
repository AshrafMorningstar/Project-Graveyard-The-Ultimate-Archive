# DevBeats - Ashraf Morningstar Edition

This is a music streaming interface clone designed for developers, customized for **Ashraf Morningstar**.

## Features

- **Dark Mode Aesthetic**: Inspired by Spotify, tailored for coding sessions.
- **Interactive Player**: Functional play/pause, progress bar simulation, and volume interactions.
- **Responsive Design**: Adapts to different screen sizes.
- **Custom Branding**: Personalized with Ashraf Morningstar's profile.

## Deployment

This website is built with vanilla HTML, CSS, and JavaScript, making it extremely easy to deploy to static hosting services like Netlify or Vercel.

### Deploy to Netlify

1. **Drag and Drop**: Log in to [Netlify](https://app.netlify.com/), and drag the `Spotify` folder onto the dashboard.
2. **Git Integration**: Push this repository to GitHub, link it in Netlify, and it will auto-deploy.

### Deploy to Vercel

1. **Install Vercel CLI**: `npm i -g vercel`
2. **Deploy**: Run `vercel` in the project root.
3. **Git Integration**: Similar to Netlify, connect your GitHub repo to Vercel for continuous deployment.
