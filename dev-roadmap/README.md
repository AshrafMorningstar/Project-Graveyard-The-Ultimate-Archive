## üõ†Ô∏è Setup & Installation

This repository follows a monorepo structure. You can view any frontend by opening its `index.html`. However, **Tier 2 and Tier 3 projects require backend dependencies** to function fully.

### Quick Start (Windows)

Run the automated installer script to fetch all dependencies for Node.js and Python projects:

```powershell
.\install_all.bat
```

### Manual Usage

1.  **Frontend**: Simply open `index.html` in any project folder.
2.  **Backend (Node)**:
    ```bash
    cd 02-intermediate/02-booking-system
    npm install
    node server.js
    ```
3.  **Backend (Python)**:
    ```bash
    cd 02-intermediate/01-ai-summarizer
    pip install -r requirements.txt
    python app.py
    ```
