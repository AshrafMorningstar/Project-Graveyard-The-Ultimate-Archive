# Privacy-First Analytics Dashboard

**Author:** Ashraf Morningstar  
**GitHub:** [https://github.com/AshrafMorningstar](https://github.com/AshrafMorningstar)

## Overview

A lightweight, privacy-respecting analytics dashboard that demonstrates GDPR-compliant metrics visualization without third-party trackers or cookies.

## Key Features

- **No Tracking:** Client-side aggregation only
- **GDPR Compliant:** No personal data collection
- **Real-time Charts:** Chart.js visualizations
- **Clean UI:** Professional dashboard design

## Tech Stack

- Vanilla JS
- Chart.js
- CSS Grid
