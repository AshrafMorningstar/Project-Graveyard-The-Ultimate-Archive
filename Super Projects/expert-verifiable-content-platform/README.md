# Verifiable Content Platform

## Overview

Developer-facing platform to publish verifiable static pages to IPFS with optional blockchain anchoring so content authenticity can be audited. Turn any static page into a verifiable artifact.

## Tech Stack

- Frontend: React
- Backend: Node.js
- IPFS: IPFS Daemon / Infura
- Blockchain: Ethereum (optional)
- CLI: Commander.js
- Styling: CSS3

## Key Features

- Publish flow: bundle ‚Üí IPFS ‚Üí pin ‚Üí anchor
- Verification UI for third parties
- Content hash confirmation
- Digital signature support
- Link resolver
- DNSLink vanity aliasing
- Provenance timeline
- Tamper-evident design

## Setup Instructions

```bash
npm install
npm run dev
```
